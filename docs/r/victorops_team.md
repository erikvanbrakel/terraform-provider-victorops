# victorops_team
Provides a VictorOps Team.

## Example Usage
```hcl
resource "victorops_team" "my_ops_team" {
    name = "Operations"
}
```

## Argument reference
 - `name` - The name of the team. This is required.

## Attributes reference
 - `is_default_team` - Self-explanatory.
